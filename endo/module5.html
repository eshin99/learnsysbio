Module 5: t[1] (initial thyroid hormone concentration)

Digital Endocrinology by UCONN CSMLab



  1. Make t[1] equal to 0 (line 7)
  2. Press [Run]
  3. Repeat (try 5, 15, 25, and 30 for t[1])
N = 100; // total number of data points r = new Array(N); // thyrotropin-releasing hormone (TRH) concentration t = new Array(N); // thyroid hormone (TH) concentration R = 10; // constant thyrotropin-releasing hormone concentration r[1] = R; // initial thyrotropin-releasing hormone concentration t[1] = 1; // initial thyroid hormone concentration p_rt = 0.7; // production/secretion parameter p_t = 0.7; // degradation/excretion parameter for (n = 2; n < N+1; n++) // N-1 loops { r[n] = R; // constant thyrotropin-releasing hormone concentration t[n] = p_rt*r[n-1] + p_t*t[n-1]; // update thyroid hormone concentration } // end of for loop
