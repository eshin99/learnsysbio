Module 5: y[0] (initial y protein concentration)

Learn Systems Biology by UCONN CSMLab



  1. Make y[0] equal to 0 (line 9)
  2. Press [Run]
  3. Repeat (try 5, 15, 25, and 30 for pY)
N = 100; // total number of data points x = new Array(N); // x protein concentration (uM) y = new Array(N); // y protein concentration (uM) X = 10; // constant x protein concentration x[0] = X; // initial x protein concentration y[0] = 1; // initial y protein concentration pXY = 0.7; // production parameter pY = 0.7; // degradation/dilution parameter for (n = 1; n < N; n++) // N-1 loops (1 loop = 1 min) { x[n] = X; // constant x protein concentration y[n] = pXY*x[n-1] + pY*y[n-1]; // update y protein concentration } // end of for loop
